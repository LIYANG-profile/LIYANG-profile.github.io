# 个人作品集网站 · 建设规划

> 状态：已确认视觉方向，待素材齐备后开发
> 设计参考：`design-ref/reference-homepage.png`（浅色编辑杂志风）

---

## 一、项目目标

- **用途**：求职作品集（视频后期 / 账号运营岗位）
- **核心叙事**：内容创作（剪映 + 达芬奇）× 数据运营（Excel + Power BI）
- **硬性要求**：
  1. 网络长期可访问（≥ 2 个月），电脑 + 手机双端适配
  2. 通过 5 个视频作品展示 3 类能力
  3. 有审美和动效

## 二、作品与能力结构

| 能力类别 | 作品数 | 说明 |
|----------|--------|------|
| 前期 · 脚本拍摄 | 1 | 展示脚本策划与拍摄执行能力 |
| 后期 · 影片前瞻 | 2 | 知识分享类，影片前瞻推荐 |
| 后期 · 财经口播 | 2 | IP 人设、短平快财经知识 |

技术栈标签：剪映、DaVinci Resolve、Excel、Power BI

## 三、视觉基调（依据参考图）

- **风格**：浅色编辑杂志风（Swiss Editorial）
- **色板**：
  - 背景：暖白 `#FAF8F4`
  - 主文字：近黑 `#111111`
  - 强调色：橘红 `#E8442E`（标题笔刷下划线、悬停态、小标签）
  - 辅助灰：`#8a8a84`（次级文字、编号、分隔线）
- **字体**：中文黑体大标题（粗、紧排）+ 英文小号无衬线辅助标签（大写、宽字距）
- **版式特征**：
  - 细分隔线构成的网格版式
  - 区块编号（01 / 02 / 03…）+ 左侧竖排 "PORTFOLIO" 文字
  - 中英双语标签混排（如「精选作品 SELECTED WORKS」）

## 四、页面结构（单页滚动）

依照参考图框架，从上到下：

1. **顶部导航**：左侧 Logo（姓名/IP 名 + 英文身份副标），右侧锚点导航（作品 / 能力 / 关于 / 联系）
2. **01 · Hero 首屏**：
   - 问候语「你好，我是 ×××」
   - 大标题「内容创作 × 数据运营」（首词橘红笔刷下划线）
   - 两行副文案
   - 右栏数据摘要（经验年数 / 作品数据等，数字滚动动效）+「查看简历 →」
3. **02 · 精选作品（滚动视频墙，核心区）**：
   - 视频卡片横向自动滚动（无限循环跑马灯）
   - **电脑端**：悬停 → 滚动暂停、该卡片视频静音播放；移开 → 恢复滚动；点击 → 详情弹层
   - **手机端**：轻触 → 暂停并播放；再次轻触 → 恢复；支持手指拖动
   - 卡片构成：封面图 + 时长角标 + 分类彩色小标签 + 标题 + 职责标签行
   - 详情弹层：完整片段播放 + 四段式说明（背景 → 我的角色 → 工具 → 结果）+ 完整版外链（可选）
4. **03 · 三类能力分区**：三块能力卡片，各自关联对应作品，展示工具图标
5. **04 · 数据运营能力**：Excel / Power BI 成果图表展示 + 数据叙事
6. **05 · 关于我 + 联系方式**：简介、微信 / 邮箱、简历下载（可选）
7. **页脚**：版权 / 所在地 / 社交链接

## 五、技术方案

| 项 | 选择 | 理由 |
|----|------|------|
| 框架 | Vite + React + TypeScript | 纯静态输出，AI 开发效率高 |
| 样式 | Tailwind CSS | 快速实现编辑风网格版式 |
| 动效 | Framer Motion | 滚动渐入、卡片悬停、数字滚动 |
| 视频 | 自托管 30 秒 MP4 片段（H.264，720p，单个 ≤10MB） | 免外部依赖，总体积约 30–50MB |
| 性能 | 封面图 poster + 懒加载 + `preload="metadata"` | 首屏不下载视频 |
| 部署 | GitHub Pages（用户已有账号），Vercel 备用 | 免费、无到期 |
| 内容管理 | 作品信息集中在 `src/data/works.ts` 单文件 | 日后加作品只改一处 |

## 七、开发阶段

1. **搭建**：初始化 Vite 项目 + Tailwind + Framer Motion
2. **结构**：按第四节完成全部区块（占位素材）
3. **视觉**：按第三节还原参考图气质（网格线、编号、双语标签、笔刷下划线）
4. **动效**：滚动视频墙交互（悬停暂停/播放、移动端轻触）、滚动渐入、数字滚动
5. **适配**：手机端布局（卡片纵向堆叠或单列滚动）、真机验证视频播放
6. **替换素材** → **部署 GitHub Pages** → **双端验证**
7. **交付**：公开网址 + 维护说明（如何加作品、如何重新发布）

## 八、验收标准

- [ ] 电脑 / 手机均可正常访问与播放视频
- [ ] 视频墙交互符合第四节定义
- [ ] 首屏加载不下载视频文件
- [ ] 视觉还原参考图基调，动效流畅
- [ ] 公开 URL 可分享给 HR
